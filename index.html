./cornell

Bringing Big Red App back to the 80s

View the Project on GitHub mrkev/cornell

Install

If you have node.js: 

npm install -g cornell

If you dont have node.js:

curl https://raw.githubusercontent.com/mrkev/cornell/master/lib/install.sh | bash

Usage

  • cornell --help - Always a good idea
  • cornell eat - Let's you pick a hall, view the menu.
  • cornell eat okies - Okenshields' menu.
  • cornell class mexico - Search roster for classes containing "mexico".
  • cornell class -s CS compilers - Search CS classes containing compilers.
  • cornell class -s CS -l "%subj%cnum %when" | grep spring - Fun stuff.

Wait but why.

  1. You already have 42 tabs open and don't want a new one just to check menus.

  2. You're in bash already anyways ¯\_(ツ)_/¯

  3. You want to tmux and cron and grep for dayz.

  4. Cause Back in the 80s life was good. The grass was green, UIs didn't suck and kids weren't whining about their scrolling animations or their OTA software updates.

  5. To up your vintage game. You're welcome.
